## 部署方法

### Worker（明文源吗）
1. https://dash.cloudflare.com/workers
2. 揀 cfnew worker → Edit Code
3. 全選 → 刪除 → 粘貼 明文源吗 完整內容
4. Save and Deploy

### Wrangler CLI
```bash
cd ~/.kimaki/projects/cfnew-local
npx wrangler login   # 第一次要授權
npx wrangler deploy
```
